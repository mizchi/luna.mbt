# APG Pattern References

WAI-ARIA Authoring Practices Guide (APG) patterns reference documentation.

These files are sourced from [masuP9/apg-patterns-examples](https://github.com/masuP9/apg-patterns-examples) and provide detailed implementation guides for accessible UI components.

## Contents

Each `.md` file contains:
- ARIA Requirements (Roles, Properties, States)
- Keyboard Support
- Focus Management
- Test Checklist
- Implementation Notes

## Usage

Reference these documents when implementing or improving headless components in `src/x/components/`.
