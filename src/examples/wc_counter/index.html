Implementation

This counter is implemented in MoonBit and compiled to JavaScript.

The component uses:

  • @mizchi/luna-wcr - JavaScript runtime for Web Components
  • @mizchi/stella - CLI for generating Web Component wrappers
  • stella/component - MoonBit context API
