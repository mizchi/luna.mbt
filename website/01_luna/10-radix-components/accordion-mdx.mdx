## Key Points

1. **MDX Component**: `<accordion-ssr>` is a custom element (Web Component)
2. **SSR Content**: HTML inside the tag is rendered at build time
3. **Hydration**: `accordion-ssr.js` attaches event handlers without replacing DOM

## Comparison: Markdown vs MDX

| Feature | `<Island>` (Markdown) | `<custom-element>` (MDX) |
|---------|----------------------|--------------------------|
| SSR children | Not preserved | **Preserved** |
| Hydration | Full re-render | Event handlers only |
| File extension | `.md` | `.mdx` |
