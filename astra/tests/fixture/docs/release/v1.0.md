# Release v1.0

A page whose URL segment contains a dot (`/release/v1.0/`). The build
crawler must route this through `url_to_output_path` (writing
`release/v1.0/index.html`), not treat it as a flat asset file. Tests
that the asset-vs-page split uses membership in the asset URL set
instead of a "last segment contains a dot" heuristic.
